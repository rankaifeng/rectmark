这是一个基于react开发的图像标注程序,目前里面只支持矩形的标注。

## 启动流程

在这个项目中你可以这样启动:

### 1. `npm install 或者 yarn`
### 2.`npm start 或者执行 yarn start`
打开浏览器 [http://localhost:3000](http://localhost:3000)

### `npm run build`

 这个打包发布！

### `npm run eject`
当然你也可以看到npm的全部配置信息！
